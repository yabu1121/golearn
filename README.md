# Go言語 完全学習ガイド

このリポジトリは、Go言語の基礎から実践的なWebアプリケーションバックエンド開発まで、体系的に学習できる教材です。

## 📚 学習の進め方

各章は独立したディレクトリになっており、順番に学習することをお勧めします。
各ファイルは実行可能で、コメントで詳しく解説しています。

```bash
# 各ファイルの実行方法
go run <ファイル名>.go

# 例
go run 01_basics/01_hello.go
```

## 🗂️ カリキュラム構成

### 1. 基礎編 (`01_basics/`)
- 変数と定数
- データ型
- 演算子
- 制御構文（if, for, switch）
- 配列とスライス
- マップ
- ポインタ

### 2. 関数とメソッド編 (`02_functions/`)
- 関数の基本
- 複数戻り値
- 可変長引数
- 無名関数とクロージャ
- 構造体
- メソッド
- インターフェース
- エラーハンドリング

### 3. 並行処理編 (`03_concurrency/`)
- Goroutine
- Channel
- Select文
- Mutex
- WaitGroup
- Context
- 並行処理パターン（ファンアウト/イン、ワーカープール、パイプライン）

### 4. 標準ライブラリ編 (`04_stdlib/`)
- JSON処理
- HTTP クライアント
- ファイルIO（読み書き、ディレクトリ操作）
- 時刻処理（フォーマット、Timer/Ticker）

### 5. Webバックエンド基礎編 (`05_web_basics/`)
- HTTPサーバーの基本
- ルーティング
- ミドルウェア
- テンプレート
- フォーム処理
- セッション管理

### 6. データベース編 (`06_database/`)
- SQLite基礎（接続、CRUD操作）
- トランザクション
- プリペアドステートメント
- エラーハンドリング

### 7. REST API編 (`07_rest_api/`)
- RESTful API設計
- CRUD操作の完全実装
- 認証システム（簡易JWT）
- バリデーション
- ページネーション
- フィルタリング
- エラーハンドリング
- CORS対応

### 8. 実践プロジェクト (`08_project/`)
完全なWebアプリケーションバックエンド
- ユーザー管理API
- 認証システム
- データベース統合
- ミドルウェア実装
- テスト

## 🚀 環境セットアップ

### 前提条件
- Go 1.21以上がインストールされていること

### 確認方法
```bash
go version
```

### 依存パッケージのインストール
```bash
# プロジェクトルートで実行
go mod init learn-go
go mod tidy
```

## 💡 学習のポイント

1. **コードを実際に書く**: 読むだけでなく、必ず自分で書いて実行してください
2. **エラーを恐れない**: エラーメッセージは学習の宝庫です
3. **公式ドキュメントを読む**: https://go.dev/doc/
4. **小さく始める**: 各章を確実に理解してから次に進みましょう
